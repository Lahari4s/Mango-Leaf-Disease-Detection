# 🥭 Mango Leaf Disease Detection

## 📌 Overview
This project detects mango leaf diseases using deep learning models such as CNN, EfficientNet, and ResNet50.

## 🚨 Problem Statement
Farmers struggle to identify crop diseases early, leading to reduced yield.

## 💡 Solution
A deep learning model that classifies leaf images into disease categories.

- CNN / EfficientNet / ResNet50
